# LunaAI

## LunaAI, a Multi-Lingual Voice Agent that engages in conversation, answering queries with ease and with accuracy and provide call specialists with a dashboard to consolidate info.

# Built With
- Frontend
  - Next.js
  - Tailwind CSS
- Backend
  - Twiml
  - Amazon Polly
  - FastAPI
  - Pinecone
  - OpenAI
